# Repo Notes

This repository is a small agent playground. Keep it friendly for future
agents that need to read code, make a focused change, run tests, and draft a
pull request.

- Prefer deterministic fixtures and tests.
- Avoid adding external dependencies unless the exercise specifically needs
  them.
- Keep new agent exercises small enough to complete in one PR.
- When changing behavior, update the matching tests and docs.
